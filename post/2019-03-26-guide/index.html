American National Election Studies (ANES) 2016

Useful tool to explore variables in ANES dataset based on categories

  • SDA utility from UC Berkeley
    • I recommend this as tool to explore which variables are available, and which ones are measuring similar concepts. This is useful when you are not very familiar with the ANES data, and not sure exactly what data are available.
    • For visualization and analysis, I recommend using statistical softwares (Stata, R etc.) for reproducibility reasons.
